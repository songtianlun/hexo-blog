## 选择题笔记
* 1966年Boehm和Jacopini证明了程序设计语言仅仅使用顺序、选择和重复三种基本控制结构就足以表达出各种其他形式结构的程序设计方法。
* 实体型强调数据类型，实体集强调数据值
* 在关系模型中，域的含义就是属性的取值范围。
* 在ALTER TABLE命令中使用CHANGE能够更改字段名和数据类型。
* MySQL中增加列的基本语法：
```
ALTER TABLE 表名 ADD [COLUMN] 列名 数据类型 约束条件
```
* UNIQUE 唯一约束可以有多个唯一约束，且可设置在表的多列上
